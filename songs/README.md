# Songs Directory

This directory contains bhajans and songs for the Sathi-AI music player.

## File Naming Convention

- Use lowercase letters
- Replace spaces with underscores
- Examples:
  - `radhe_shyam.mp3`
  - `hanuman_chalisa.mp3`
  - `om_jai_jagdish_hare.mp3`

## Supported Audio Formats

- MP3 (preferred)
- WAV
- M4A
- FLAC
- OGG

## Adding Songs

1. Place your audio files in this directory
2. Rename them following the naming convention
3. The music player will automatically detect them

## Example Commands

- "play radhe shyam bhajan"
- "play hanuman chalisa"
- "start music"
- "stop music"
